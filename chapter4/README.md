# Web開発新人研修 第4章のサンプルアプリ
[WEB+DB PRESS Vol.92 Web開発新人研修 第4章](http://gihyo.jp/magazine/wdpress/archive/2016/vol92)のサンプルコードです。

## サンプルアプリの起動方法
```
bundle install
npm install
npm run build
bundle exec rake db:migrate
bundle exec rails server
```
